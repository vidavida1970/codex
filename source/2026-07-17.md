# ChatGPT로 성공하기｜Day 03

- 발행일: 2026-07-17
- 원본 Site: https://addit-chatgpt-success-day1.vidavoda.chatgpt.site/
- 제목: 모델과 노력 수준을 업무에 맞게 고르는 법

## 오늘의 학습목표

업무의 난이도·속도·오류 비용을 평가해 GPT‑5.6 모델 티어와 노력 수준을 선택합니다.

## 핵심 개념

- Luna: 짧은 초안, 형식 변환, 반복 문구처럼 검수가 쉬운 작업
- Terra: 자료 요약, 여러 안 비교, 일반적인 기획·분석 작업
- Sol: 복잡한 제안 논리, 다단계 분석, 오류 비용이 큰 최종 결과물
- 노력 수준: 같은 모델이 문제를 얼마나 깊게 탐색·검토할지 정하는 설정. 제공 범위는 제품과 요금제에 따라 다릅니다.

선택 공식: 오류 비용 낮음·검수 쉬움 → 빠르게 ｜ 비교·논리 필요 → 균형 ｜ 심사·의사결정·최종 납품 → 깊게

## 잘못된 사용법과 개선

모든 SNS 문구에 가장 강한 모델과 최고 노력 수준을 쓰는 대신, 빠른 티어로 초안을 만들고 균형 티어로 압축한 뒤 강한 티어와 높은 노력으로 최종 후보를 심사표에 맞춰 검토합니다.

## 오늘의 실제 사례

Model ML은 2026년 7월 GPT‑5.6 조기 평가에서 20개 고난도 고객 워크플로와 수백 개 덱을 비교했습니다. 회사 측은 GPT‑5.6 Sol이 비교 모델보다 덱당 토큰을 39% 적게 쓰면서 더 정돈된 레이아웃과 명확한 데이터 시각화를 만들었다고 보고했습니다. 이는 OpenAI 공식 발표에 포함된 고객 평가이며 독립 검증 수치는 아닙니다.

국내 마케팅·공공기관 제안 현장에서 Sol·Terra·Luna를 같은 과제로 비교하고 측정 기준과 결과를 공개한 검증 가능한 사례는 확인하지 못했습니다.

## 복사형 실전 프롬프트

```text
다음 업무에 적합한 GPT‑5.6 모델 티어와 노력 수준을 추천하라.

[업무]
{예: 공공기관 제안서의 핵심 홍보전략 1페이지 작성}

[판단 정보]
- 결과 사용처: {내부 아이디어/고객 검토/최종 제출}
- 오류 발생 시 영향: {낮음/중간/높음}
- 작업 복잡도: {낮음/중간/높음}
- 사람이 검수하기 쉬운가: {예/아니요}
- 원하는 완료 시간: {분 또는 시간}

[출력]
1. 추천 모델 티어: Luna/Terra/Sol 중 1개
2. 추천 노력 수준: 내 화면에서 선택 가능한 범위 안에서 제안
3. 선택 이유 3개
4. 더 가벼운 대안과 사용 조건
5. 결과 검수 기준 3개

현재 제품 화면에서 확인되지 않는 설정명은 만들지 말고, 선택 항목이 보이지 않으면 ‘현재 화면에서 사용 가능 여부 확인’이라고 표시하라.
```

## 결과 체크

- 업무 난이도보다 틀렸을 때의 비용을 먼저 고려했는가?
- 초안 단계와 최종 검토 단계를 분리했는가?
- 수정 횟수·오류·기준 충족률로 품질을 측정할 수 있는가?

## 오늘의 미션

애드아이티 업무 3개를 초안·분석·최종 납품으로 분류하고 각각 Luna·Terra·Sol 중 우선 후보를 정하세요. 선택 이유와 검수 지표를 한 줄씩 붙이면 완료입니다.

> 최고 설정을 고르는 사람이 아니라, 오류 비용에 맞춰 자원을 배분하는 사람이 AI로 일을 완성합니다.
